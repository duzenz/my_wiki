# Disk Usage Commands

```bash
du -sh folder_name
```

```bash
dh -f
```

# Tar File
```bash
tar czf output_name.tar.gz zipped_folder_name/
```

# Untar File
```bash
tar -xvzf output_name.tar.gz
```

```bash
sudo dmidecode --type 17
```
